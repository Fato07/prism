The OpenAPI source for the public reporting API lives in the docs app:

```txt
apps/docs/openapi/prism-public-api.yaml
```

## Endpoints

| Method | Path | Purpose |
| --- | --- | --- |
| `GET` | `/api/public/stats` | Aggregate public Prism metrics. |
| `GET` | `/api/public/history` | Recent validated traces, clamped by `limit`. |
| `GET` | `/api/public/traces/{id}/report` | Full Prism Report for one trace. |

## Try with curl

```bash
curl https://prism-dashboard-production-e6e3.up.railway.app/api/public/stats

curl "https://prism-dashboard-production-e6e3.up.railway.app/api/public/history?limit=2"

curl https://prism-dashboard-production-e6e3.up.railway.app/api/public/traces/d6cdd60f-f5e0-43ab-ba2d-7dcab76a8e24/report
```

## MCP sentinel note

The sentinel validation endpoint is intentionally not described as REST OpenAPI. It is MCP JSON-RPC/SSE at:

```txt
https://prism-sentinel-production.up.railway.app/mcp/
```

Use the [x402 + MCP validation guide](./x402-mcp-validation) for that flow.
