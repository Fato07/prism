Prism validates agent reasoning before it becomes market action. A trader agent creates a Trading-R1 trace, an adversarial sentinel challenges it, and the result is exposed as a Prism Report with deterministic reasoning metrics and receipt links.

## What you can do here

- Run the Prism CLI without cloning the whole repository.
- Quote and call the sentinel's x402-protected MCP endpoint.
- Read public dashboard APIs for stats, history, and trace reports.
- Verify IPFS CIDs, Base Sepolia payment transactions, Arc validation receipts, and calibration evidence.
- Understand the safety rules around Circle wallets, Polymarket token IDs, and explicit paid flows.
